# ⚽ Football Analytics Dashboard

## 📊 Overview

This project is an interactive football data analysis dashboard built with Streamlit.
It allows users to explore player performance, filter teams, and visualize basic key statistics of 4 known players.

## 🚀 Features

* Upload custom football datasets (CSV)
* Filter players by team
* Key performance metrics (goals, assists, goals per 90)
* Data visualizations (distribution, rankings, scatter plots)

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib
